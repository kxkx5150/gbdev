GB ASMチュートリアル

Hello World!

このレッスンでは、まず最初のプログラムを組み立てます。 この章の残りは、どのように、そしてなぜ動くのかを説明することに専念します。

たくさんのコマンドを入力する必要があるので、今すぐターミナルを開いてください。 新しいディレクトリを作成するのが良いでしょう（例えば mkdir gb_hello_world とし、cd gb_hello_worldで新しいディレクトリに入る）。

以下のファイルを取得し（各リンクを右クリックし、「名前を付けてリンク先を保存」）、すべてこの新しいディレクトリに配置します。

  • ヘロヘロワールド.asm
  • 株式会社ハードウェア

そして、やはりそのディレクトリ内のターミナルから、次の3つのコマンドを実行します。

⚠️

個々のコマンドがどこから始まるかを明確にするために、各コマンドの前に$を付けましたが、タイプしないでください!

rgbasm -L -o hello-world.o hello-world.asmrgblink -o hello-world.gb hello-world.orgbfix -v -p 0xFF hello-world.gb

‼️

引数の扱いに注意ここでの-oのようなオプションは、その後ろの引数をパラメータとして使用するものがあります。

  1. rgbasm -L -o hello-world.asm hello-world.oは動作しません (hello-world.asm が破損する可能性があります!)。
  2. rgbasm -L hello-world.asm -o hello-world.oは動作します。
  3. rgbasm hello-world.asm -o hello-world.o -Lでも動作します。

引数内に空白が必要な場合は、引用符で囲む必要があります。

  1. rgbasm -L -o hello world.o hello world.asmは動作しません。
  2. rgbasm -L -o "hello world.o" "hello world.asm"は動作します。

このように表示されるはずです。

(自分では解決できないエラーが発生した場合は、遠慮なくお問い合わせください。解決いたします)

おめでとうございます！これでゲームボーイのROMを初めて組み立てました！あとは実行するだけです。 BGBを開いて開いたウィンドウを右クリックし、hello-world.gbをロードしてください。

🤕

If you are using Wine and BGB is failing to start (or gives a blank/black screen), try these troubleshooting tips.

And in any case, don’t hesitate to ask for help!

You could also take a flash cart (I use the EverDrive GB X5, but there are plenty of alternatives), load up your ROM onto it, and run it on an actual console!
